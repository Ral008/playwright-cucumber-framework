# Playwright Cucumber Framework

## 🚀 Stack
- Playwright
- Cucumber (BDD)
- TypeScript
- Allure Reports
- Docker

## ▶️ Run
- npm install
- npm run test

## 📊 Report
- npm run report
- npm run report:open
